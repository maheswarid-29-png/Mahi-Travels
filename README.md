# Mahi Travels Website
This is a beginner-friendly sample frontend travel website created for a student project.  
It is designed to help users explore destinations, view packages, and book travel easily.

## Website Type
Travel Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub (for hosting)
- Netlify (optional deployment)

## Pages Included
- Home
- About
- Destinations
- Packages
- Gallery
- Booking
- Contact

## Features
- Fully responsive design (mobile, tablet, desktop)
- SEO-friendly pages (meta tags included)
- Travel destination gallery with images
- Package listing with cost calculator
- Booking form with JavaScript validation
- Contact page with business details
- Mobile navigation menu
- Active page highlighting in navigation

## How to Run
1. Download or clone the project folder
2. Open `index.html` in any browser
3. Navigate through all pages using the menu

## GitHub Pages Deployment (Optional)
1. Create a GitHub repository
2. Upload all project files
3. Go to Settings → Pages
4. Select branch: `main`
5. Click Save
6. Your website will be live at a GitHub URL

## Netlify Deployment (Optional)
1. Login to Netlify
2. Click "Add new site"
3. Choose "Import from GitHub"
4. Select your repository
5. Deploy the site
